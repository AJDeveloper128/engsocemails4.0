Want project experience? Leadership roles? CoCreCo does a few amazing things!

  • We run the textbook rental program, offering textbooks at affordable costs to students.

  • A team for developers (C.ENG, ES&C, CompSci & S.ENG) that runs in subteams of Database & Cloud, Website, and Machine Learning. Now working with a $20,000 Machine Learning Server on campus.
  • A mechanical engineering project team (Creating automated textbook rental lockers)
